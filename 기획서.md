# 팜유 (FarmU)

**제11회 농업·농촌 공공데이터 + AI 활용 창업경진대회 기획서**

> **아이디어명:** 팜유 (FarmU) — 조합원 성과율 통합 분석부터 작목 개선 실행까지 연결하는 농업 조합 특화 AI 플랫폼

---

## 서비스 한 줄 정의

> 조합원의 운영성과(생산·출하·수익)를 **하나의 성과율(%)**로 표준화하고,  
> 저성과 원인을 **필지·작목 적합도로 진단**하여,  
> **개선 실행(작목선택·출하전략·멘토링)**까지 연결하는 조합 특화 AI 플랫폼

---

## 1. 기획 개요

농업 조합의 성과 평가는 여전히 담당자 경험에 의존하고, 출하 시점은 감에 기대며, 저성과 원인은 짐작으로 처리된다. 데이터는 있지만 연결되지 않는다.

팜유(FarmU)는 조합원의 생산·출하·수익을 **하나의 성과율(%)**로 표준화하고, AI가 저성과 원인을 설명한 뒤, 작목 전환·출하 전략·멘토링까지 **실행으로 이어주는** 농업 조합 특화 AI 플랫폼이다.

같은 데이터 위에서 운영 책임자는 **조합 전체를 관리**하고, 조합원은 **본인 실행 리포트**를 받으며, 역할에 따라 화면과 권한이 완전히 분리된다.

## 2. 제안 배경

농업 조합·법인의 경영관리는 현재 세 가지 구조적 단점을 안고 있다.

| 단점 | 현황 | 결과 |
|---|---|---|
| **평가의 단점** | 성과 평가가 암묵·자기 중심적, 담당자 역량에 따라 편차가 크다 | 비교 불가 |
| **대응의 단점** | 가격·기상·임금 변동성은 커졌지만 데이터 기반 체계가 없다 | 사후 대응 |
| **전략의 단점** | 단기 이익(출하)과 중장기 전략(작목)이 별도 업무로 분리되어 있다 | 진단이 실행으로 안 이어짐 |

> "평가 → 진단 → 실행"을 **한 흐름으로 연결**하는 플랫폼이 필요하다. 팜유는 이 단점을 메우는 데서 출발한다.

## 3. 기존 서비스와의 차별성 및 독창성

### 3-1. 기존 서비스 대비 위치

| 구분 | 기존 서비스 | 한계 | 팜유 |
|---|---|---|---|
| 영농기록·경영장부 앱 | 농가 개인 기록 중심 | 조합 단위 통합·비교 불가 | **조합 표준 성과율로 통합 비교** |
| 농산물 가격정보 서비스 | 시세 조회 제공 | "그래서 언제 출하?"의 답 못 함 | **출하시점·리스크 액션 매핑** |
| 스마트팜 관련 솔루션 | 실내 환경 모니터링 | 노지·경종·경제 성과와 분리 | **성과율↔필지 적합도 연결** |
| 컨설턴트(오프라인) | 전문가 진단 | 비용·시간·표준화 한계 | **XAI 자동 진단 + 표준 리포트** |

### 3-2. 5대 차별성

| # | 차별성 | 한 줄 정의 |
|---|---|---|
| ① | 조합 표준 성과율(%) 제공 | 생산·출하·수익 가중 통합, 가중치 공개로 조합원 간 비교 신뢰성 확보 |
| ② | XAI 기반 저성과 원인 진단 | SHAP으로 항목별 기여도 도출, "우선 개선 3과제" 형태로 제공 |
| ③ | 필지 적합도 기반 개선 제안 | 저성과 원인을 토양·기상·지형과 결합해 작목 전환안 + 예상 성과 변화 제시 |
| ④ | 단기 출하 + 중장기 작목전략 통합 | 출하 적중률(단기)과 수익성(중장기)을 하나의 의사결정 흐름으로 연결 |
| ⑤ | 조합·농가·컨설턴트 통합 플랫폼 | 같은 데이터 위에서 역할별 화면·권한을 완전히 분리 |

> **독창성 핵심:** 공공데이터(가격·기상·필지·토질)를 **조합 내부 데이터(재배·출하·판매)와 결합**해, "평가용 지표"가 아닌 **"실행 가능한 개선안"**으로 출력하는 것.

## 4. 농림축산식품 공공데이터 활용

성과평가(가격·기상·수급) · 필지 적합성(토양·지형·일조) · 조합 내부(재배·출하·판매) 데이터를 결합해 **"평가용 지표"가 아닌 "실행 가능한 개선안"**으로 출력한다.

| 데이터 | 출처 | 활용 |

| 출하물 가격·등급·이력 | [EKAPE](https://www.ekape.or.kr) | 경락가격·등급편차 분석 |
| 기상·농업기상 | [기상청 기상자료개방포털](https://data.kma.go.kr) | 작황 영향 변수 |
| 필지경계 (팜맵) | [농림부 스마트팜맵](https://www.farmmap.kr) | 필지 단위 분석, 지도 시각화 |

| 생산·출하 통계 | [농림부 공공데이터포털](https://data.mafra.go.kr) | 조합 성과 벤치마킹 |
| 지형(경사·고도) | [국토지리정보원 DEM](https://www.ngii.go.kr) | 필지 적합도 보정 |
| 지역통계 | [통계청 KOSIS](https://kosis.kr) · [SGIS](https://sgis.kostat.go.kr) | 지역 작목·농가 구조 분석 |
| 조합 내부 데이터 | 조합 ERP·엑셀 | 재배·출하·판매 이력 |

## 5. 구체성 (구현·실현 가능성)

### 성과율 산식

`조합 성과율 = 생산성(35) + 출하의사결정(35) + 수익성(30)` — 파일럿 1개월 동일가중치, 이후 조합 합의 가중치 분기 조정

### AI 모듈

| 모듈 | 한 줄 정의 |
|---|---|
| 출하시점 예측 | 회귀·시계열로 권고 출하 구간 및 리스크 액션 도출 |
| 리스크 스코어링 | 가격·기상·수급 변동 → 출하 보류/분할/작목 재검토 자동 매핑 |
| 성과 원인 설명 | SHAP 기반 항목별 기여도 → "우선 개선 3과제" 출력 |
| 작목 적합도 스코어링 | 필지×작목 매칭 점수 → 추천 랭킹 + 예상 성과 변화 제시 |

### 단계적 도입

`1단계` 엑셀 업로드 → `2단계` 반자동 정기 연동 → `3단계` 조합 ERP·공공데이터 API 실시간 연동

## 6. 시장성 및 사업화 가능성

### 고객 & 시장

전국 지역 협동조합 약 **1,100여 개** (출하·영농·원예조합 등) — 월별 평가·보고·지도업무가 상시 발생해 구독형(SaaS)에 최적

### 비즈니스 모델

| 모델 | 내용 | 성격 |
|---|---|---|
| ① 조합 SaaS 연 구독 | 기본 성과율 대시보드 | 안정적 기반 매출 |
| ② 농가 프리미엄 리포트 | 작목 적합도·선택 분석 | 확장 매출 |
| ③ API 연계 | 지자체·금융·보험 연동 | 고마진 확장 |

### 사업화 로드맵

| 단계 | 기간 | 목표 |
|---|---|---|

## 8. 기대효과

### 주체별 효과

| 주체 | 효과 |
|---|---|
| 조합 | 의사결정 속도·질 향상, 평가 표준화, 보고 자동화 |
| 농가 | 출하 타이밍·작목 선택 최적화로 수익 개선 |
| 지역·정책 | 데이터 기반 작목선택·지원사업 설계 근거 확보 |

### 정량 KPI

- 조합 평균 성과율 **+10%p** — 통합 성과율 도입 후 조합원 평균 기준
- 출하 적중률 **+12%** — AI 권고 채택 건 대비 실제 이익 발생 비율
- 보고서 작성시간 **↓70%** — 수작업 대비 자동 생성 시간 기준
- 작목추천 적용 필지 성과 **+8~10%** — 전환 후 동일 필지 전년 대비
- 저성과 조합원 비중 **↓15%** — 성과율 하위 20% 구간 농가 비중 감소

### 파급효과

산업 — 권역 간 성과 비교·벤치마킹 체계 형성 / 민간 — 금융·보험·유통의 신용평가·리스크 계산 활용 / 정책 — 지역 단위 지원사업 근거 기반 설계

## 10. 사용자 권한 체계

총 3개 역할. 컨설턴트 업무는 운영 책임자(UNION_ADMIN)가 겸한다.

| 역할 | 명칭 | 진입점 | 주요 권한 |
|---|---|---|---|
| `SUPER_ADMIN` | 시스템 관리자 | `/admin` | 전체 조합·계정·공지·로그 관리 |
| `UNION_ADMIN` | 운영 책임자 | `/dashboard` | 조합원 관리, 컨설팅 겸임, 데이터 업로드·가중치 설정 |
| `MEMBER` | 조합원 | `/me/analysis` | 본인 데이터 열람, 출하·시나리오·리포트 개인 한정 |
