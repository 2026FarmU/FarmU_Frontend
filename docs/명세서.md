## 2.4 권한

> 역할은 **총 2개**. 컨설턴트 업무는 조합 관리자(UNION_ADMIN)가 겸한다.

### UNION_ADMIN (조합 관리자 · 운영 책임자)

* 조합 단위 운영 책임
* 조합원 데이터 열람·관리
* **컨설턴트 역할 겸임** — 조합원 상담, 멘토링 승인, 시나리오 조언
* 데이터 업로드·가중치 설정 권한

### MEMBER (조합원)

* 본인 데이터만 열람
* 출하·시나리오·리포트 본인 한정 사용
* `/me/analysis` 진입점

# 9. 공통 에러 처리

## HTTP 정책

### 401

* refresh 시도
* 실패 시 logout

### 403

* /403 이동

### 404

* EmptyState 표시

### 500

* Toast 표시
* Sentry 전송

---

## 대표 에러 메시지

| Code                | Message                |
| ------------------- | ---------------------- |
| INVALID_CREDENTIALS | 아이디 또는 비밀번호가 일치하지 않습니다 |
| WITHDRAWN_USER      | 탈퇴한 회원입니다              |
| FILE_TOO_LARGE      | 파일이 너무 큽니다             |
| INVALID_PASSWORD    | 기존 비밀번호가 일치하지 않습니다     |

---

# 10. EmptyState 정책

| 페이지       | 문구             |
| --------- | -------------- |
| Dashboard | 현재 위험 알림이 없습니다 |
| Members   | 조회 결과가 없습니다    |
| Shipping  | 출하 권고가 없습니다    |
| Lands     | 등록된 필지가 없습니다   |
| Reports   | 생성된 리포트가 없습니다  |

# 11. actionUrl 정책
